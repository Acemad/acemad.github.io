Install Instructions
  • Unzip the downloaded file somewhere in your computer.
  • Copy the theme-named folder, for example : NeonRift, to ux0:VitaShell/theme/.
  • Switch to the new theme from VitaShell settings. Restart VitaShell. Enjoy.
  • (Optional) If you would like to install the LiveArea theme, bubble and boot screen (availability depends on each theme), just copy the VITASHELL folder to ur0:appmeta/ and ux0:app/, and make sure to confirm any overwrite.
